üìñ Game Rules (Simplified for AI)

üéØ Objective:

  • Move your rhombus (‚óÜ) to opponent's base (Black: 3-0, White: 3-8)
  • Alternative wins: Checkmate opponent's rhombus, draglock, repetition

üé≤ Piece Types & Movement:

Piece Symbol Movement
Square ‚ñ† Adjacent hexes (6 directions)
Triangle ‚ñ≤ 3 forward directions (orientation-dependent)
Rhombus ‚óÜ 2 forward directions + diagonal (special)
Circle ‚óè Perimetric (zone edges) + gateway transitions
Hexagon ‚¨° Adjacent + rotation (orientation-dependent)

‚öîÔ∏è Combat Rules:

  • Pieces capture by moving to occupied hex
  • Exception: Rhombus cannot capture rhombus
  • Captured pieces are eliminated

üö´ Special Rules:

  • Base Defense: Rhombus at base blocks opponent's win
  • Dead Zone (3-3, 3-4, 3-5): Circles cannot move from here
  • Threefold Repetition: Same position 3x = loss
  • Fivefold Shuttle: Same shuttle pattern 5x = loss
  • Draglock: No legal moves = loss
  • Escape Race: When only rhombuses remain, base defense disabled

üìù RPN Move Notation System

Standard Move Format:

FROM‚ÜíTO [CAPTURE] [ROTATION]

Examples:
"3-0‚Üí3-1"          // Rhombus moves from 3-0 to 3-1
"2-2‚Üí3-3 C"        // Triangle captures at 3-3
"4-4 R+"           // Hexagon rotates clockwise
"2-3‚Üí3-4 C R+"     // Capture + rotate

Coordinate System:

Hexagonal grid with row-column notation: row-column

  • Black starts at top (row 0)
  • White starts at bottom (row 6)
  • Black goal: 3-0
  • White goal: 3-8

Piece Notation:

Code Piece Color
SQ-W Square White
TRI-B Triangle Black
RHO-W Rhombus White
CIR-B Circle Black
HEX-W Hexagon White

Modifiers:

  • C - Capture
  • R+ - Rotate clockwise
  • R- - Rotate counter-clockwise
  • ! - Check (rhombus threatened)
  • # - Checkmate
  • WIN - Winning move

üí° Best Practices for AI/LLM Players

Strategic Guidelines:

  • Early Game: Develop pieces, control center
  • Mid Game: Create threats, capture valuable pieces
  • Late Game: Push rhombus, support with tactics
  • Piece Values: Triangle (180) > Hexagon (170) > Circle (160) > Square (150)
  • Rhombus: Keep safe, advance only when supported

Common Pitfalls to Avoid:

  • ‚ùå Don't move rhombus too early without support
  • ‚ùå Don't ignore threats to your pieces
  • ‚ùå Don't create repetition patterns (3x = loss)
  • ‚ùå Don't shuttle pieces back and forth (5x = loss)
  • ‚ùå Don't forget base defense (keep rhombus at base early)
